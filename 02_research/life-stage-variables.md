## 調査日：2026年6月3日（read-only web research）

対象：**シロザケ＝白鮭 / *Oncorhynchus keta***、北海道で人工ふ化放流される個体を中心に。
用途：Adobe Fresco で手描きするデータビジュアライゼーション（生活史×変数）の数値根拠。

## 3. 確度フラグ・注意

- **スピード**：成魚のみ実測（Tanaka et al. データロガー）。幼稚魚・未成魚は生物学的推定 → 図では破線＋「推定」表記にする。
- **深さ・水温**：種全体の値が中心で、3段階に厳密分割した一次値は乏しい。
- **回帰率**：地域差・年変動が大きく近年低下。見出しは ~4%、近年は 2〜3% と併記。
- **ふ化率 ~95%**（卵→ふ化）と、お手本の文脈にある「卵→稚魚 80〜90%」は**測る区間が違う**（飼育減耗まで含めれば後者も妥当）。
- 未成魚の体長 20〜50cm は明示的一次値ではなく、放流5cm→回帰65cm の漸増からの補間。

## 4. 図的表現の設計メモ（各変数をどう描くか）

データビジュアライゼーションは「**左＝川・幼稚魚 → 中＝外洋・未成魚 → 右＝回帰・成魚**」という横方向の旅（journey）を背骨にし、7変数を上下のトラックで重ねる小複数型を想定。

- **期間（月）** … 背骨そのもの＝**横軸（時間）**。月メモリを振り、各段階の帯幅で滞在期間を表す（沿岸の1〜2か月は狭く、海洋の3〜5年は広く、河川の2〜3週は極端に狭く）。時間の非線形さ（海が圧倒的に長い）が一目で出る。
- **大きさ** … 背骨の上に**実比のサケシルエット**を3つ。5cm→~30cm→65cm を相対スケールで（成魚は幼稚魚の約13倍）。サイズ差そのものがメッセージ。
- **スピード** … 各段階の上に**矢印**。長さ＝速度。成魚=実線（2km/h・実測）、幼稚魚/未成魚=破線＋「推定」。
- **深さ** … 背骨から**下方向の縦帯**。縦軸に水深メモリ（0/60/250m）。幼稚魚=表層の細帯、外洋=0〜60mの帯＋250mまでの点線（最大潜行）、成魚=表層。
- **水温** … 各段階に**色帯ストリップ**（寒色→暖色グラデ）。5〜13 / 8〜12 / 0〜24°C を帯の幅・位置で。氷 `#C8E0E8`→teal→赤系で温度感を直感化。
- **摂餌** … アイコン＋頻度。幼稚魚=小さな餌粒（体重3%/日）、未成魚=小魚・プランクトン群、成魚=**✕「絶食」**で断絶を強調。
- **生存率** … **減っていく点の漏斗**。放流100点 → ふ化~95% → 沿岸で大量減耗 → 回帰 ~2〜5%。点の数が劇的に減る視覚で「100尾→2〜3尾」を表現。

凡例（段階カラー＝幼稚魚 teal明 `#6EB0C4` / 未成魚 teal `#2E6E8E` / 成魚 赤 `#A83232`）と出典脚注を下部に。

## 5. 出典

- **水産研究・教育機構 (FRA)** — サケの放流数と来遊数及び回帰率／SALMON情報 No.4（三陸沿岸 サケ稚幼魚の食性）
- **北海道立総合研究機構 (HRO) さけます・内水面水産試験場** — サケ人工種苗生産／来遊予測
- **水産庁・農林水産省 (MAFF)** — 「サケは春夏秋冬どこにいますか」（回帰率3〜5%・季節別海洋回遊ルート）
- **岩手県水産技術センター** — ミニ知識/サケ（回帰サイズ65cm/3.7kg、ふ化率95%、回帰率4%）
- **群馬県水産試験場** — サケ科魚類の卵からの飼い方（給餌量 体重3〜4%）
- **Tanaka et al.** (PubMed 11807107) — 成魚の遊泳速度・遊泳深度（データロガー実測 1.5〜3.0 km/h、満月時に浅く速い）
- **Cdn. J. Fish. Aquat. Sci. (2023)** — 稚魚の代謝特性（機能適温 8〜12°C）
- **FishBase**（O. keta, ID 241）— 体長・体重・深度(0〜250m)・水温(0〜24°C)・栄養段階(3.7)
- **NOAA / ADF&G / Wikipedia** — 成魚の年齢・絶食・食性・体重幅
