# この制作物の意図と制約（評価時に踏まえてほしい前提）
- 目的：最終ビジュアルの「方向性」を4案で比較検討するための内部プロトタイプ（完成作品そのものではない）。
- 品質目標：ctxt.jp（文脈デザイン研究所「和食人類学」）水準のエディトリアル品質。
- 重要な制約：制作者（AI）は新規に作図していない。ネット上の信頼できる実素材だけを収集・評価・配置し、テキスト注釈で構成している（＝キュレーション手法）。
  - 再利用素材：CC0 / PD / CC-BY（出典・ライセンス明記で転用可）
  - 着想用の既存事例：著作権物は「参照のみ・転用不可」としてテキスト引用
- 採用デザインシステム（ctxt.jp由来）：地＝paper #FAFAF6／図版パネル＝warm-paper #F4F0E6＋teal #2E6E8E の2pxトップルール／角丸0／和文＝Noto Serif JP・欧文/数字＝EB Garamond／英語グロス・◇ディバイダ・脚注体の出典表記。

# 4つの方向（ボード）
- A 回遊ジャーニーマップ：太平洋中心のLAEA（正積方位図法・淡色）地図上に、回遊ルート・3段階・変数を注釈。
- B 放射状ライフサイクル時計：PDの円環ライフサイクル図に「弧長＝滞在期間」等を対応。
- C 横断トラック（小複数）：時間の横軸に7変数を段で重ねた自社スケッチ図（最も網羅的）。
- D 生存の漏斗×ジャーニー：筋子（約3,000卵）→放流→回帰2〜3尾の激減を、Minard/Isotype/Sankeyを参照しつつ表現。

# 扱っているデータ（3段階 × 7変数。信頼性も評価対象）
- 段階：幼稚魚（当歳・沿岸）／未成魚（外洋1〜3年）／成魚（回帰3〜5歳）
- 期間：沿岸1〜2か月／海洋3〜5年／河川2〜3週間
- 大きさ：5cm・1g → ~30cm（漸増）→ 65cm・3.7kg（最大100cm/18kg）
- スピード：~1〜3体長/秒(推定) ／ 不明(推定) ／ 1.5〜3.0km/h≒55cm/s（実測）
- 深さ：表層 ／ 0〜60m（最大250m）／ 表層
- 水温：放流適温5〜13℃ ／ 機能適温8〜12℃ ／ 種の耐性0〜24℃
- 摂餌：体重3%/日(動物プランクトン) ／ 小魚・カイアシ(栄養段階3.7) ／ 入河後は絶食(0)
- 生存率：ふ化~95%(人工) ／ 海洋減耗 ／ 放流→回帰 約2〜4%（近年低下）
- 確度注記：スピードは成魚のみ実測（他は推定）、深さ・水温は種全体の値、回帰率は地域差・年変動あり
- 主な出典：水産研究・教育機構(FRA)／北海道立総研機構さけます試験場／水産庁・農水省／岩手県水産技術センター／群馬県水試／Tanaka et al.(遊泳ロガー実測)／FishBase

# 評価してほしい軸（各10点満点。根拠・具体例を必ず添えること）
1. エディトリアル/デザイン品質：タイポグラフィ階層、余白とリズム、配色、構図、図版の枠取り、一貫性。ctxt.jp水準との距離を具体的に。

3. 情報の網羅性：データの過不足、欠けている観点、注釈の十分さ。
4. 情報の信頼性：データ値と出典の妥当性、確度注記の適切さ、誤り・要再検証点の指摘。
5. 出典・ライセンス処理の厳密さ：転用可/参照のみの区別、クレジットの正確さ、権利上のリスク。
6. コンセプト適合：「収集素材のみで作図しない」という制約の達成度と、それが品質に与えた功罪。
