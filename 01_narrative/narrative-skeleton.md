# 物語骨格：サーモンは遡上する
## 「日本のサーモン生態」グラフィックエディトリアル

---

## 構造原理

「遡上」を文字通りの構造として使う。  
読者は上流（現代・表層）から下流（記憶・深層）へではなく、  
**下流（消費の現在）から上流（記憶の源流）へ**と進む。

魚が川を遡るように、読者も時間を遡る。

---

## Act 1：消費の現在

**場面**  
2025年。日本のどこかの回転寿司店。サーモンのにぎりが流れてくる。  
現在、サーモンは日本の寿司ネタ消費量第1位（くら寿司・スシロー調査より）。

**問い**  
なぜこれが当たり前になったのか。いつからそうなのか。

**主要な事実**  
- 1985年以前、日本の寿司に「生サーモン」は存在しなかった  
- 当時の日本のサケ（白鮭）には顎口虫など寄生虫のリスクがあり、生食の習慣がなかった  
- 「サーモン」はカタカナで書かれる——それは外来語であり、外来の魚である
- 2023年時点で日本はノルウェーのサーモン輸出先として世界13位、アジア3位（中国・韓国に次ぐ）、年間24,000トン・29億6,000万NOK（出典：Seafood Norway）

**グラフィックの対応**  
→ 回転寿司のサーモンの皿の断面図（アップ写真 or 解剖図的イラスト）  
→ 「サーモン消費量の推移グラフ」（1980〜2025、折れ線、1985年前後で急変）  
→ タイポグラフィ：「サーモン」vs「シャケ」の二語が並ぶ

---

## Act 2：渡来の物語

**場面**  
1985年。ノルウェー水産省と日本の商社の間で繰り広げられた、長い交渉。  
「Project Japan」（ノルウェー政府公認のサーモン対日輸出推進事業）。  
日本の寿司職人は最初、生サーモンを拒否した。

**問い**  
どうやってこの魚は「外来種」から「定番」になったのか。  
ノルウェーの養殖フィヨルドと、北海道の天然川は何が違うのか。

**主要な事実**  
- 1985年〜1995年の10年間にわたる交渉と試食キャンペーン  
- アトランティックサーモン（大西洋サーモン）は養殖環境が閉鎖系のため寄生虫リスクが低い  
- ノルウェーは当時、石油産業と並ぶ輸出産品として養殖サーモンを国家戦略に位置づけた  
- 「サーモン寿司」は日本発信ではなく、ノルウェー発信の発明だった

**グラフィックの対応**  
→ 「ふたつの地図」：北太平洋の回遊ルートとノルウェー→日本の輸送航路を同一投影で重ねる  
→ タイムライン：1985「Project Japan開始」→ 1986「試食拒否」→ 1995「スーパー定着」  
→ 養殖フィヨルドの断面図と北海道の川の断面図の対比（同縮尺）

## Act 3：川に戻る論理

**場面**  
10〜11月。北海道・千歳川。  
産卵のために川を遡上するシロザケの群れ。水面が割れる。

**問い**  
なぜ魚は必ず生まれた川に戻るのか。  
サケが死んだあと、その栄養はどこへ行くのか。

**主要な事実**（検証済み数字あり）
- シロザケ（白鮭）は北海道の主要な遡上種。生まれた川の「においの記憶」で帰還する
- 産卵後に死んだサケの体は川岸の森に栄養（海洋ミネラル）を運ぶ——熊、鳥、木が食べる
- 「サケが来ない川に豊かな森はない」という生態系連鎖
- **千歳川捕獲数**：2022年ピーク587,475尾 → 2024年161,276尾（2年で72.6%減）→ 2025年12月時点67,283尾（記録的低水準）
- **北海道日本海地区**：2022年607万尾 → 2024年119万尾（**1990年以降最低**）
- 出典：千歳水族館公式統計・水産研究教育機構2025年報告書（いずれも一次データ）

**グラフィックの対応**  
→ 川の縦断面図：源流（産卵床）→中流（若魚期）→河口（海への旅立ち）→外洋回遊→帰還  
→ 生態系連鎖の図（サケ→熊→糞→森→ミネラル→川→サケ）  
→ フィールドワーク写真：千歳川・遡上するサケ  
→ 遡上数の変化データ（主要河川別、水産庁統計）

## Act 4：神の魚

**場面**  
秋、北海道のアイヌのコタン（集落）。  
「カムイノミ」——神への感謝の儀礼。  
サケは「カムイチェップ（神の魚）」と呼ばれる。

**問い**  
アイヌにとってサケとは何か。  
「食べる」という行為は、何を意味していたか。

**主要な概念**（検証済み）
- **カムイチェップ（kamuy-cep）**：神の魚。サケは神の化身であり、人間に食べられることで神の世界に帰る（相互贈与）
- **シペ（si-pe）**：「真の食べ物」。アイヌにとってサケは主食以上の存在
- 全身利用：頭・身・卵・白子・胃袋・骨・内臓・目まで。皮は靴（**チェップケリ**）・衣服・帽子に
- 先祖供養（**イチャルパ**）の供物：**チマチェップ**（焼いたサケ）
- **アシリチェプノミ**：秋の初遡上を迎える感謝の儀礼（ヌサ＝祭壇に供物、カムイノミ）
- 明治期の段階的漁業規制（1876年〜1902年）でアシリチェプノミは消滅
- **1982年**に札幌・豊平川で約100年ぶりに復活、**1991年**に千歳でも別途復活
- 2019年アイヌ新法：「先住民族」認定だが漁猟・採集権は保障されず
- **2020年**：ラポロアイヌネイションが浦幌十勝川でのサケ漁業権を求め提訴（2024年一審敗訴・控訴中）

**グラフィックの対応**  
→ アイヌ文様のサケの図像（伝統的な装飾スタイルで描いた魚の解剖図的表現）  
→ 「カムイチェップ」の概念図：人間↔神の往還の構造  
→ 地図：アイヌの生活圏とサケ漁場の重なり（江戸期）  
→ アーカイブ写真または現代コミュニティのフィールドワーク記録

## Act 5：帰着

**場面**  
冒頭と同じ回転寿司の皿が戻ってくる。しかし見え方が変わっている。

**問い（の解体）**  
「これは何の魚か」に対する答えは、一つではない：

- ノルウェーのフィヨルドで育ったアトランティックサーモン（輸入）
- 北海道の川で生まれ外洋を回遊して戻るシロザケ（遡上）
- アイヌにとって神の化身であった魚（カムイチェップ）
- そのどれもが「日本のサーモン」になった

**結語（候補）**  
「シャケは川を遡って生まれた。サーモンは海を渡ってやってきた。  
どちらも今、同じ皿の上にある。」

**グラフィックの対応**  
→ 5つのActを一枚の「川の断面図」に統合した総括インフォグラフィック  
→ 「シャケ」の江戸図鑑様式と「サーモン」の現代写真を並置  
→ テキスト：最小限

## 構造サマリー

```
[現在] 回転寿司のサーモン
    ↑
[1985] Project Japan / 渡来の物語
    ↑
[生態] 千歳川 / 遡上する論理
    ↑
[深層] アイヌ / カムイチェップ
    ↑
[帰着] 「これは何の魚か」の複数の答え
```

読者の動線はこの矢印の逆——下から上ではなく上から下へ遡る。  
Webではスクロール方向と「遡上」感の演出が要検討。
